# 🎵 猜歌游戏插件

群聊听歌识曲互动游戏，支持网易云音乐热门中文歌曲。

## 功能特点

- 🎶 从网易云热门歌单随机选取歌曲（仅保留2-7字纯中文歌名）
- 🔊 发送语音片段，群友抢答
- � 会话制游戏，发送 🎶 加入（支持中途加入）
- ✨ 答对自动进入下一轮
- �💡 提示功能（逐步揭示歌名）
- 🏆 排行榜功能
- 🎭 游戏结束真心话/大冒险

## 命令列表

| 命令 | 说明 |
|------|------|
| `#猜歌` | 创建游戏 |
| `🎶` | 加入游戏 |
| `#开始猜歌` | 开始游戏 |
| `#猜歌提示` | 获取歌名提示 |
| `#猜歌答案` | 公布答案（管理员） |
| `#猜歌排行` | 查看历史排行榜 |
| `#猜歌退出` | 结束游戏 |
| `#猜歌帮助` | 显示帮助信息 |

## 玩法说明

1. 发送 `#猜歌` 创建游戏
2. 群友发送 `🎶` 加入（可中途加入）
3. 发送 `#开始猜歌` 开始
4. 直接发送歌名抢答
5. 答对自动进入下一轮
6. 达到轮数或发送 `#猜歌退出` 结束

## 配置项

| 配置 | 说明 | 默认值 |
|------|------|--------|
| `round_timeout` | 每轮超时时间（秒） | 60 |
| `max_rounds` | 最大轮数 | 10 |
| `min_players` | 最少参与人数 | 1 |
| `max_players` | 最多参与人数 | 20 |
| `voice_send_timeout` | 语音发送超时（秒） | 15 |
| `admin_ids` | 管理员QQ号列表 | [] |

## 安装

1. 将插件文件夹放入 AstrBot 插件目录
2. 安装依赖：`pip install aiohttp`
3. 重启 AstrBot
